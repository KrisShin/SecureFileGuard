## 5. 配置文件
```yaml
app:
  name: “项目名称”
  version: “版本号”
  author: “作者”
  release_time: ”发布时间”
path:
  db_file: “数据库文件路径”
  upload: “默认文件管理路径”
  download: “默认下载路径”
security:
  default_algorithm: “默认算法AES”
  algorithms:
    - “AES”
    - ”DES/3DES”
    - ”SM4”
other:
  其他需要存储的内容
```
